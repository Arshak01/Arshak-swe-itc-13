# Sorting array of strings application

## There are some cases which sorting must correspond

```
Ունենք 3 մասից բաղկացած տող՝ <key> <part> <part>։
<key>֊ը ունի հստակ կառուցվածք՝ սկզբում տառ + թիվ (w1, er3, t456 ...)
<part>-ը ունի հետևյալ կառուցվածքները՝ միայն տառեր(ert), միայն թվեր(45), տառ + թիվ(w23), բայց ոչ թիվ + տառ(4w5 կամ 7y):
Ներմուծել n քանակի տող և սորտավորել հետևյալ կետերով՝
	-Սկզբում տեղակայվում են տառային տողերը, հետո թվայինները (թվային տող է համարվում այն տողը, որի <part>֊երը բաղկացած են միայն թվերից՝ (w1 456 7)-թվային  (ds3 23 w3)-տառային );
	֊Տառային տողերը սրտավորել հետևյալ կերպ՝
		-Առաջին հերթին պետք է նայել <key>-ի տառային մասին՝ սորտավորել ըստ ASCII-ի
		֊Տառային մասի կրկնության դեպքում նայել <key>-ի երկարությանը
		֊Երկարության նույնության դեպքում դասավորել ըստ <key>-ի թվային մասի (թվանշաններով)
		
		Եթե <key>֊երը ամբողջությամբ նույնն են դասավորել ըստ <part>֊ի՝
		
		֊Առաջին հերթին պետք է նայել տառի պարունակությանը՝ եթե մի մասում տառ կա իսկ մյուսինում՝ ոչ, տառ պարունակողը բարձրանում է վերև (we3 f45 fr4 > we3 12 f45)
		֊Հետո՝ <part>-ի երկարությանը
		֊Հետո դասավորել ըստ տառային մասի ASCII-ի
		֊Հետո նոր թվային մասի (թվանշաններով)
	֊Թվային տողերը սորտավորել հետևյալ կերպ՝
		֊Առաջին հերթին նայել <key>-ի թվային մասին (թվանշաններով)
		֊Հետո <key>-ի երկարությանը
		֊Հետո <key>-ի տառային մասին ըստ ASCII-ի
		
		Եթե <key>֊երը ամբողջությամբ նույնն են դասավորել ըստ <part>֊ի՝
		
		֊Առաջին հերթին պետք է նայել <part>-ի երկարությանը
		֊Հետո նոր թվային մասին
```		
Օրինակ՝


```
Input:
	wz3 45 df8
	qe3 12 907
	wz3 f12 mn0
	wz3 f12 mn1
	asd45 df 77
	po12 567 907
```

```
Output:
	asd45 df 77
	wz3 f12 mn0
	wz3 f12 mn1
	wz3 45 df8
	po12 567 9
	qe3 12 907
```
